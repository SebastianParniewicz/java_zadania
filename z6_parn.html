Odpowiedzi do zadań

  • Rodzaje okienek:
    • Okienko alert – wyświetlająca okienko dialogowe z informacją i przyciskiem OK.
    • Okienko prompt – generująca okno, w którym użytkownik może wpisać wartość, zwracaną następnie do programu.
    • Okienko confirm służy do tego, aby użytkownik coś potwierdził (stąd nazwa confirm).
  • Konwersja liczb:
    • Wczytanie zmiennej z użyciem `prompt` wykonywane jest jako ciąg znaków dlatego konieczna jest konwersji z ciągu znaków na wartości liczbowe aby wykonywać obliczenia.
    • parseFloat(zmienna) - konwersja zmiennej do typu float.
    • parseInt(zmienna) - konwersja zmiennej do typu int (liczbowego).
    • toString(zmienna) - konwersja zmiennej do typu łańcuchowego.
  • Przy nadawaniu nazw zmiennym należy przestrzegać pewnych zasad:
    • Nazwa zmiennej musi się zaczynać od litery, znaku "$" lub "_", może zawierać tylko litery (bez polskich znaków), cyfry oraz "$" i "_".
    • Wielkość liter ma znaczenie.
    • Należy nie używać słów kluczowych JavaScript jako nazw zmiennych.
  • Typy zmiennych w JavaScript:
    • Number - Typ liczbowy, reprezentuje liczby całkowite i zmiennoprzecinkowe.
    • String - Typ tekstowy, reprezentuje ciągi znaków.
    • Boolean - Typ logiczny, reprezentuje wartości logiczne true (prawda) lub false (fałsz).
    • Null - Pusta zawartość.
    • Undefined - Typ niezidentyfikowany.
    • Object - Obiekt, ogólny typ danych.
